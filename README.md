# Идентификация параметров моделей ползучести с помощью нейронных сетей

Решение обратной задачи механики вязкоупругости, обучение нейронной сети для восстановления трех физических параметров ($a_0, a_1, a_2$) по кривым ползучести.

## Структура проекта

* `1_base_experiment.ipynb` — базовый численный эксперимент (100 образцов, временная сетка из 5 точек, 1% гауссова шума).
* `2_improved_experiment.ipynb` — оптимизированный эксперимент с уплотненной временной сеткой (8 точек) и увеличенной выборкой (200 образцов).
* `3_no_noise_experiment.ipynb` — эксперимент на точных данных без добавления измерительного шума для определения предела точности архитектуры.

## Данные
Сгенерированные датасеты в формате CSV (`creep_dataset_physical.csv`, `dataset_improved.csv`, `experiment1_no_noise.csv`) создаются автоматически при запуске соответствующих jupyter-ноутбуков.
