# Идентификация параметров моделей ползучести с помощью нейронных сетей

Решение обратной задачи механики вязкоупругости, обучение нейронной сети для восстановления трех физических параметров ($a_0, a_1, a_2$) по кривым ползучести.

## Структура проекта

* `base_experiment.ipynb` — базовый численный эксперимент (100 образцов, временная сетка из 5 точек, 1% шума).
* `improved_experiment.ipynb` — улучшенный эксперимент (200 образцов, временная сетка из 8 точек, 1% шума.
* `no_noise_experiment.ipynb` — эксперимент на точных данных без добавления измерительного шума.
